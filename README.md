# 🇫🇷 טיול דרום צרפת — מדריך אינטראקטיבי

מדריך משפחתי לטיול בדרום צרפת (פרובאנס + ריביירה), קיץ 2026.
דף HTML בודד, בעברית RTL, mobile-first, עם מסלול יום-אחרי-יום, לינקים חיים ל-Google Maps, מסעדות, אוכל, היסטוריה, טיפים ואזהרות.

## מבנה התיקייה

```
France-Trip/
├── index.html          ← הדף הראשי (כל התוכן + העיצוב + הלוגיקה בקובץ אחד)
├── images/             ← תמונות אמיתיות של היעדים
│   ├── provence/       ← תמונות פרובאנס (גורד, רוסיון, ורדון...)
│   ├── riviera/        ← תמונות ניס / קוט דאזור
│   └── shared/         ← תמונות כלליות (אוכל, באנרים, אווירה)
└── README.md
```

## הרצה מקומית

לפתיחה פשוט פותחים את `index.html` בדפדפן.
לתצוגה כמו שרת (מומלץ בשביל לבדוק נתיבי תמונות):

```bash
cd "France-Trip" && python3 -m http.server 8000
# ואז פותחים http://localhost:8000
```
